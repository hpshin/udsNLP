Course description

자연언어처리와 이해에서는 자연언어처리의 기본 개념과 방법론을 학습한다. 특히 텍스트 처리에 초점을 맞추어, raw text를 처리하는 다양한 방법들과  특히 최근에 많이 활용되고 있는 deep learning 기법을 이용한 언어처리와 vector semantics에 기반한 방법론에 대해서도 학습한다. python기반의 keras를 활용하여 한국어와 영어의 문서분류, 언어모델링, 텍스트 생성을 집중적으로 실습하도록 한다. 또한  Natural Langauge Processing with Python을 중심으로  자연언어처리를 위한 python toolkit도 실습하도록 한다. 영어자료 처리 외에 한국어 자료 처리와 관련한 여러 이슈들도 다루어진다.
